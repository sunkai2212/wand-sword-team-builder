## 状态与可访问性

- 统一 CSS tokens 管理背景、表面、文字、边线、accent、危险色和危险边框。
- 缺省技能和宠物使用中性虚线，不表达为错误状态。
- 主操作、选中、焦点、禁用、加载和危险操作拥有一致视觉语言。
- 所有过渡只使用 `opacity` 与 `transform`，时长 160ms。
- `prefers-reduced-motion: reduce` 下关闭非必要过渡和按压位移。
- Dialog 标题、取消、焦点限制与返回保持原有正确行为。
- aria-live、图片 alt、aria-pressed 和 disabled 语义未回归。
